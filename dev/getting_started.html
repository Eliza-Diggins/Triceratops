Quickstart Guide#

Welcome to TRICERATOPS! This guide will help you get started with the code, including installation instructions and resources for new users and developers.

Installation#

Triceratops is currently in development and has not been released on PyPI. To install Triceratops, you can clone the repository from GitHub and install it using pip. Here are the steps:

  1. Clone the Triceratops repository:

    git clone https://github.com/eliza-diggins/Triceratops

  2. Navigate to the cloned directory:

    cd Triceratops

  3. Install Triceratops using pip:

    pip install -e .

This will install Triceratops in editable mode, allowing you to make changes to the code and see them reflected immediately.

Important

In the future, Triceratops will be available on PyPI, and you will be able to install it using:

pip install triceratops

Getting Started#

To get started using Triceratops, we recommend the following steps:

  1. Look through the documentation available at Triceratops User Guide to familiarize yourself with the relevant modules and the general structure of the library. This documentation is written with brevity in mind to ensure that you can quickly find the information you need and get started with your work.

  2. Explore the examples provided in the Triceratops Example Gallery document. These examples will cover most of the basic functionality of Triceratops and will help you understand how to use the library effectively.

If you have specific questions about using Triceratops, encounter a bug, or have suggestions for improvements, please feel free to open an issue on the Triceratops GitHub repository. We welcome contributions and feedback from the community to help improve the library.

Developers#

Are you interested in helping to develop Triceratops? We welcome contributions from developers of all skill levels. To get started as a developer, please refer to the Triceratops Developer Guide for information on how to contribute, coding standards, and the development workflow.
